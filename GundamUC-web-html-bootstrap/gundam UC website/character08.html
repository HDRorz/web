巴纳吉·林克斯【BANAGHER LINKS】（バナージ·リンクス）
CV：内山昂辉
年龄：16岁
本作的主人公，卡帝亚斯·毕斯特亲生儿子，因其母不愿意他卷入毕斯特家族的命运在其年少时由其母亲带领离开毕斯特家族前往其它移民星系居住。后因为母亲的死亡，被未曾相见过的父亲安排进入阿纳海姆下属学校学习。后与神秘少女奥黛莉·本（Audrey Burne）（米妮瓦·扎比）相遇，卷入了“拉普拉斯之盒”的事件之中。最后的神棍结局中精神脱离了肉身与精神感应结合（就是进化到了不需要肉体的纯思维生命形态），但是最后自己认为“不能放弃巴纳吉·林克斯这个肉体”而恢复了原貌。

年龄：112岁
比斯特财团的创始人,也是整部小说引发事件的罪魁祸首.塞亚姆在年轻的时候(UC纪元之前)只是一个普通的为了家人 的生活而到处去工作的年轻人.一次偶然的维修卫星工作的机会让他卷入“拉普拉斯事件”(小说的序章,OVA动画第1话开头的回忆录).当炸毁了"拉普拉 斯"卫星后.他们所在的运输船也被该事件的幕后黑手抹杀,由于他所在的地方刚好没有被运输船的爆炸所牵连的太大.侥幸活了下来.在宇宙中漂浮的时候,塞亚 姆无意中得到了关键东西."拉普拉斯之箱".事后被救的塞亚姆靠"拉普拉斯之箱"从此在黑道和暗中创业.后期著名的军火商人阿纳海姆便是他的杰作.年幼时 期的塞亚姆看过了殖民卫星坠落地球的悲惨事件没有勇气去打开"拉普拉斯之箱"将其中的内容公告天下.当财团和自己的事业壮大起来之后.塞亚姆选择将自己冷 冻睡眠以借此延长寿命;当0096年他苏醒过来之后.塞亚姆决定让卡帝亚斯找到合适的继承人.将"拉普拉斯之箱"中的内容公告天下.还给地球和宇宙居民一 个"应有的未来".交代完这一切之后.塞亚姆又再次进入了冷冻睡眠之中.EP4开头处重新出现，其中提到了继承他的巴纳吉，然后又睡着了，是永恒的睡眠 了？


  

盖尔·张【Gael Chan】（ガエル·チャン）
