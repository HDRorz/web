本网站主题为SUNRISE出品世界观为UC世纪的高达作品介绍网站。选择包括《机动战士高达》、《机动战士高达～第08 MS小队》、《机动战士高达0080～口袋中的战争》、《机动战士高达0083》、《机动战士Z高达》、《机动战士高达ZZ》、《机动战士高达～逆袭的夏亚》、《机动战士高达独角兽》8部作品作为主要内容。因为此8部作品代表了UC世纪这一世界观中主要的故事进程。
本网站作品作为计算机应用能力大赛的参赛作品，协作者为上海大学计算机学院大二的沈杨华同学。学长提供技术支持。
本网站使用了Notepad++ team开发的notepad++ v6.2.3作为主要开发软件，另外使用了Adobe Dreamwave CS5.5用于资料录入，Adobe Photoshop CS用于图片编辑，还有bootstrap开发框架。使用了HTML5、Javascript、 CSS3、jQuery和PHP。（PHP脚本由沈学长完成）
主页BGM是08MS小队的OP——暴风雨中的光辉。著名燃曲，高达作品中我最喜欢的歌之一。本页BGM是渚，出自CLANNAD。其他页面的BGM是迷之BGM。（我不会告诉你们的）

其实挑选这主题的灵感也来自于那个未知名资料站。因为那个网站太旧了，所以想做得更好。
建站花费时间略长，12年11月就开始构思和规划布局并试写主页。虽然很早就开了坑，但用时却不多，一周能有1小时思考布局和内容分布已经用时很多了。其实真正开始写是从寒假开始才开始的，在那之前已经写好了一个主页，但因学长推荐了bootstrap开发框架，于是又重新改写主页。大概经过6个版本才到了目前这个版本。因为是陌生的框架，最初使用起来困难略多，请教了学长好几次，才理解了使用用法。Bootstap优点在于预先定制了大量CSS+Javascript脚本，使用了jQuery特效，方便快速，而缺点在于可定制性略差。
虽说用时略长，但实际制作周期大概只有近30天（寒假近3周+开学后几个周末），其实也很匆忙。不过好歹还是完成了预定计划。没招美工是个大失误，美工花了我大量时间。（朋友的一个东方project介绍站坑了2年了还没坑完，不过他挖的坑太好了，一辈子都别想填坑了233）
IE大概只有9和10能正常观赏。（IE是什么，能吃吗？）
