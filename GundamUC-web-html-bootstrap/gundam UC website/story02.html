时代背景

U.C.0079年1月3日，一年战争爆发，在拥有新型兵器“机动战士(MS)”的吉翁公国军队面前，地球联邦军陷入了窘迫的境地，这种情况直至9月联邦军的MS登场方有所好转。此后，联邦军的MS大量投入战场，除了占绝大多数的RGM－79“GM”之外，亦有一小批高成本的MS，即RX－78的量产型RX-79[G]陆战用先行量产型高达，分配给地球圈各战区的精英MS小队使用……

故事情节

联邦军少尉西罗•亚马达/港译：薛罗•阿玛特（天田士郎）奉命从殖民卫星赶赴地球东南亚战场，途中其所乘的运输机遇到了正驾驶着先行量产型吉姆与吉翁军新式MS战斗的山德士，西罗为了营救战友驾驶着BALL挑战对方的MS，结果两败俱伤，他与敌机的驾驶员爱娜•萨哈林（港译：艾娜•莎哈莉）皆漂流到远离战场的战舰废墟中。为了求生存，西罗与爱娜齐心协力引爆了战舰中残留的导弹，引来了各自的救援人员。此番经历，令两人互生爱慕之意，他们原来对敌方的认识也有所改变。
降落到东南亚的西罗被分配到地球联邦军-极东方面军-机械化混成大队-MS第二中队，担任第08MS小队的队长，该小队中另有四位成员：MS驾驶员卡莲、从宇宙返回的山德士、通讯兼援护员艾雷多亚、与西罗一起降落的迈克，队伍中配备的三台MS正是RX-78GUNDAM的量产型RX-79[G]。第一次来到地球的西罗还未习惯被重力束缚的环境，就立刻投入到丛林战，在追击敌方扎古的过程中，西罗渐渐偏离了本队，与附近游击队的女孩姬奇相遇。好不容易打倒了扎古，西罗终于在自动导航器的引导下回到基地。第08MS小队接下来的任务是与游击队配合，打击敌人的大队。胜利结束了战斗后不久，却有一台巨大的MA阿普撒拉斯出现在西罗他们面前，在对方强大的火力下，08小队毫无还手之力，艾雷多亚因负伤而离开小队中。西罗却不知道，那台MA的驾驶员正是爱娜！而阿普撒拉斯乃爱娜的哥哥艾尼斯所新开发的机动装甲，现正在测试中。08小队奉命在沙漠中伏击阿普撒拉斯，激战中西罗驾驶着高达撞向对方，被带离战场。西罗及时发现爱娜就是阿普撒拉斯的驾驶员，两人再度相遇却又面临被困于冰天雪地的境地。一夜的长谈，西罗与爱娜的感情更加深了，然而立场的不同，最后两人还是相互告别，回到自己的队伍中。西罗归队后即接受军事审判，罪名是间谍罪。在军事法庭上，西罗坦言他体会到战争并不是一切，敌人也是人类，和我们一样具有感情，双方能够相互理解，不要让残酷的战争再继续下去。虽然西罗被无罪释放了，但依然受到禁闭的处罚，就在这时传来了游击队村庄被敌军占领的消息。08小队不惜违反命令，偷了大队长的空白命令书出击。西罗本想尽可能地不造成人员伤亡的情况下，让敌人投降，然而战事的发展原没有他想象的那么简单，最后西罗不得不杀死了对方的驾驶员，其耳边回响起军事法庭上将军问他的一个尖锐的问题：“西罗少尉，现在的你能够向敌人开枪吗？”
艾雷多亚终于伤愈复归，但是因为西罗擅自行动的关系，08小队被派遣到最前线、在无人支援的情况下展开侦察行动。无疑，这次名义上是要08小队戴罪立功，实际上等于要西罗他们去送死。但是，西罗的理念依然没有改变：“我一定会让大家活着回来的！能够地话，一定要在最小伤亡范围内结束战争！”战士们在战场上以自己的生命作赌注，所换来的是什么呢？U.C.0079年末，基奥军战败，在与阿普撒拉斯的战斗中，西罗与爱娜合力击败了已经疯狂了的艾尼斯。为此，西罗失去了一只手和脚，但他终于能够与爱娜在一起了。联邦军的阵亡名单上，列着西罗•亚马达少尉的名字。
战争结束后，迈克携女孩姬奇一同去寻找西罗的下落，他深信队长是不会轻易牺牲的。果然，找到了一群由西罗和爱娜命名的吉翁少年军的孩子们，经了解，西罗和爱娜还活着，只不过去了北边……
后来，在绵延不绝的山脉中，在那青树翠蔓的树林中，他们找到了西罗和已经怀孕的爱娜……
