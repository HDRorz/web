时代背景

U.C.0079年12月，由吉恩公国挑起的一年战争已进入最后的阶段，人们都已看到吉恩的日渐式微，也盼望着和谈的早日到来。然而，一部分吉恩军的顽固派，仍然执迷不悟地坚持着与联邦军的战斗
这时的联邦军，除了加紧展开最终的大攻势，亦没有放松军事技术方面的研究开发工作。由于少年英雄阿姆罗驾驶RX-78-2高达在战争中的突出表现，使联邦充分认识到MS的价值与阿姆罗所表现出的新人类能力“NEW TYPE”之重要性。为了将两者更好地结合，在地球的北极基地，联邦军试制出一架最新的RX-78高达“NT(NEW TYPE)-1阿历克斯（Alex）”。之后不久的12月21日，北极基地遭到吉恩部队的突袭。不过，幸好Alex未遭破坏，并且在掩护下被秘密送往宇宙中的中立地区SIDE6，在那里进行最后的测试。

故事介绍

宇宙历0079年8月，地球联邦军为了发挥阿姆罗•雷的NEWTYPE能力而着手开发高达阿历克斯，制式编号RX-78NT-1。12月21日，吉恩公国军的特务部队“独眼巨人”（CYCLOPS）袭击联邦军的北极基地，企图破坏Alex未果，其成员安迪•司特罗斯战死。Alex被运往宇宙，这一事件引发了之后发生在殖民卫星第6区域的一场小战争。那场战争是如此的渺小，既不影响到整个一年战争，也没有多少人知晓，但是这场口袋中的战争之残酷，却让人无法忘怀。生活在和平中的人们会麻痹自己的感觉，忘却战争的残酷与无情；好奇心旺盛的孩子们天真地玩着打仗的游戏，对于“战争”这一字眼根本不会理解它所带来巨大的痛苦与悲伤，等他们了解到时，已经失去了太多宝贵的东西。《机动战士高达0080：口袋里的战争》会告诉我们这一不为人知的小小战役，通过一个孩子的眼睛来看清那残酷的真实。

距离战场还有几英里？

阿尔弗雷德•伊兹尔哈（昵称阿尔）是位满怀好奇心的10岁少年，他与父母一起居住在殖民卫星第6区域。其父亲因工作关系长期出差在外，母亲则是位标准的家庭主妇。因为第6区域在一年战争时期保持中立，所以居住在此的人们完全感受不到外面联邦军与吉恩军之间激烈的战斗，对于已经习惯了和平宁静的生活的他们来说，战争是另外一个世界的事物。12月21日，阿尔和往常一样上学，在午餐时好友却伊拿出从他表哥处得来的联邦军肩章来炫耀，并说这是机动战士驾驶员所佩带的肩章。一旁的女同学多萝茜立刻反驳联邦军并没有机动战士，自尊心作祟的阿尔便吹嘘他老爸的关系曾亲眼见过联邦的机动战士。为了得到证据让多萝茜认输，阿尔在却伊的教唆下带着小型摄像机趁与父亲见面的机会，潜入宇宙港军管区内拍下联邦军装卸货物的情景，他并不知道在那货柜中装载的正是RX-78NT-1！在回家时，阿尔遇到了邻家的大姐克莉丝汀娜•玛肯吉（昵称克莉丝）。克莉丝是地球联邦军MS电脑测试员，因为缺少人手的关系从地球调来第6区域，担任Alex的测试驾驶员，当然这一切阿尔并不知晓。次日，正当阿尔和朋友们在学校谈论没能拍到机动战士时，卫星突然遭到吉恩军的袭击，卫星连忙出动量产型的机动战士吉姆，从未经过实战的卫星士兵们被对方的扎古打得手忙脚乱。慌乱中一架扎古的推进器被击中，掠过学校降落在远处的森林中。几分钟后，这场小规模的战斗结束了，第一次看到战争的居民们不知所措，谁也没有注意到那架坠落的扎古，更不会留心有个孩子带着摄像机奔向森林。阿尔在森林中找到那架扎古，他兴奋地对其一个劲拍摄着，摄像机的镜头由扎古的脚移到已经开启的驾驶舱，再由驾驶舱移至肩部……在扎古的肩部有一金发青年站立着，他背对着太阳，阳光将其身躯钩现出金色的边，青年举枪对着阿路。浑然不知危险是何物的阿尔仰头望着那青年，拿着摄像机的手垂了下来，这一瞬间那位青年在阿尔的心中留下了不可磨灭的印象。

茶色眼睛所映出的东西

驾驶扎古的青年名叫贝纳德•瓦兹曼（昵称巴尼），是吉恩军的新兵，此番进攻第6区域是他第一次驾驶机动战士，没想到会落得这样一个下场，再加上遇见阿尔这个好奇心旺盛的孩子，更令他头痛不已。“你是驾驶这个机动战士的人吗？那把枪是真的吗？能给我看一下吗？”阿尔围着巴尼兴奋地喊着，“这个卫星中的人从来没看见过战争吗？”巴尼不仅有些泄气，他抢过阿尔手中的摄像机观看里面拍摄的内容：“什么呀？这是我嘛，拍得不错。”镜头中出现了克莉丝，“好可爱，是你的姐姐？”接下去是阿尔在码头拍到的装载着Alex的货柜，“这是……？！”阿尔扑上去将巴尼撞倒在地，并咬了他一口夺回摄像机。突然，阿尔看到了巴尼衣领上吉恩军的肩章，他立刻象着了魔似地直盯着看，这可是花钱也买不到的东西，足够让朋友们羡慕一阵子了。看透他的巴尼用自己的肩章换来摄影机内的磁碟，跳上前来营救他的僚机匆匆离去，临行前还不忘提醒阿尔一句：“这是我们之间的秘密，不要告诉任何人！”阿尔兴高采烈地回到家，将肩章贴在衣服上模拟驾驶机动战士作战。回到基地的巴尼则因首仗的败绩而被赶到了特种部队“独眼巨人”中，顶替先前在北极战死的安迪。加上巴尼“独眼巨人”部队共有四名成员，其余三人是总是叼着香烟不放的队长修泰纳、头上绑着花头带比较好斗的加尔西亚、喜欢喝酒即便是驾驶着MS战斗时也会喝上两口的密夏。从阿尔的磁碟中，吉恩军得知Alex被运到了第6区域。在吉恩公国已经呈现败局的情况下，决不能让地球联邦军将新的MS投入战场，为此“独眼巨人”部队受命潜入卫星破坏Alex。如果在12月25日零点之前还没有完成此任务的话，吉恩军将派遣姆萨伊级轻巡洋舰用核弹摧毁此卫星！巴尼驾驶着暗藏机动战士MS-18E斗士的卡车去指定地点与队友会合。不料在途中巴尼所驾驶的货柜车差点撞到了阿尔，因此被阿尔得知“独眼巨人”部队的存在。

彩虹的尽头是什么？

阿尔坚持要求加入“独眼巨人”部队，而队长修泰纳亦利用他来刺探有关的情报，还将窃听器装入“独眼巨人”部队的肩章内交与阿尔。巴尼则受命监视阿尔，在送其回家并待阿尔母亲离开后，巴尼偷偷摸入阿尔家的宅院，不料被克莉丝误认为盗贼而将他击倒在地。巴尼和阿尔两人对克莉丝一家人谎说巴尼是阿尔同父异母的哥哥，因为是私生子的关系故不能让阿尔的母亲得晓巴尼的存在。经由这次的邂逅，巴尼与克莉丝两人互相之间有了好感。在秘密工厂内，“独眼巨人”部队正加紧装配斗士，巴尼和阿尔受命去寻找联邦军基地。在阿尔的帮助下，他们不但找到了基地，还拍摄到新型Alex的照片，得知该机动战士便是联邦军中最强的MS！送阿尔回家的巴尼又见克莉丝，克莉丝的父母似乎亦相当喜欢巴尼。12月23日，“独眼巨人”部队在卫星的几个主要场所安置了定时炸弹，还将装载着备用武器与零件的货柜车停放在不同的地方，一切都准备就绪，只等夜幕降临便开始行动。此时阿尔因斗士的驾驶员是密夏而非巴尼替后者报不平，巴尼对这个好象自己弟弟的孩子说自己已经击落了四架MS，如果再添一架便算是一流的驾驶员，还吹嘘即便对手是高达亦能轻易取胜。另一方面，联邦军对Alex的测试也将接近尾声。当克莉丝对Alex高速的反应而觉得跟不上时，一旁的同事告诉她这本就是为NEWTYPE（新人类）而开发的MS，完成之后或许会送至“白色基地”。

渡河拔树

夜幕降临，当数起爆炸在不同的地点发生后，密夏驾驶着斗士出现在街头吸引了联邦军的注意力。巴尼等人在修泰纳队长的带领下，化装成联邦的士兵潜入其秘密基地。与此同时，阿尔通过克莉丝的介绍也来到该基地参观。当巴尼三人已接近Alex时，不幸被联邦军发现。队长修泰纳在递武器给巴尼时被击中，加尔西亚不顾一切上前安放炸药时也难逃一劫。在爆炸的烟雾中，巴尼拖着昏过去的修泰纳队长逃离机库。隔窗眺望的阿尔看到一穿着红色机师战斗服的人影跳入Alex的驾驶舱，他自然没有想到那人便是克莉丝。克莉丝驾驶着Alex走出基地与斗士战斗，反过来被密夏用爆导锁命中，但是克莉丝凭藉高达外面的乔巴姆装甲而幸免于难，更用Alex双手内置的90mm加特林将斗士打成马蜂窝，密夏当场阵亡。第一次实战的克莉丝惊魂未定地呆坐在驾驶舱内，她方感受到生命这么容易会消失就昏过去了。在Alex背对的石墙后，巴尼拖着修泰纳来到一辆空置的轿车边，“战况怎么样了？那个高达呢？”修泰纳气息奄奄地问。“已经被密夏打倒了。”巴尼如是说。“谎话说得可真拙劣呀，巴尼……”修泰纳闭上眼睛撒手而去。突然间巴尼察觉到身后出现人影，连忙转身将枪口对准目标，却发现阿尔呆呆地看着他……

告诉我你在说谎，巴尼

12月24日清晨，对于卫星的居民们来说或许是难忘的一夜：突然间遭到吉恩军的袭击，卫星内许多建筑物被毁（包括阿路的学校），导致数百人的伤亡。中立卫星的居民不知道，与两军交战的主要战场所在地相比，这里的损伤实在太细微了——那里的死亡人数是以“亿”来计算的。然而，虽然在战争中人会以数量来衡量，但对于死者的亲属、友人来说，那是任何东西都无法替代的。看到从废墟下被运出来和自己一般大的孩子的尸体，阿尔只能颤抖着逃离现场。在森林中，阿尔找到了巴尼并将食物带给他：“人……真的很容易就死呢，巴尼。”“那当然！”巴尼愤慨地道，“这就是战争呀！”失去了所有“独眼巨人”部队的战友，令巴尼感到害怕和绝望，他打算逃离这个卫星。“为什么？为什么要逃？”阿尔惊讶地问，“巴尼不是最优秀的驾驶员吗？不是说可以轻易地战胜高达吗？”“那都是骗你的！”巴尼撕开修泰纳给阿尔的肩章，露出里面的窃听器给他看，说明打一开始队长他们就没有把阿尔当作伙伴，而且吉恩军将出动舰队用核弹毁灭这个卫星。“巴尼，笨蛋！”阿尔含着泪跑开。短短的几天时光，缺乏父爱的他已经将巴尼视为自己的亲哥哥，是一个值得崇拜和信赖的人物，然而这一完美的形象就这样轻易地破碎了。阿尔在回家的路上遇到了克莉丝，忍不住问她假如宇宙舰队来袭、这个卫星会毁灭时她会怎样做。“我会战斗，”克莉丝回答道，“因为战斗就会有人死去，但不战斗也是一样会死去。完全正确的事在哪儿都不存在的，只能做自己能做的事。”。不懂其含义的阿尔又一次茫然了，“等你长大后就会明白的，代我向巴尼问好。”说罢，克莉丝翩翩离去。巴尼拿着假护照来到机场，在候机厅的酒吧中他看到一个有点象克莉丝的女子。喝了杯威士忌而带点醉意的女孩走向电话亭，在电话中哽咽着骂着欺骗她的恋人。听着女子的哭声和斥责，巴尼紧握着阿尔丢还给他的肩章，深深地低下了头。躺在家中的阿尔仰望着天花板发呆的时候，母亲在楼下唤回了他的活力：“阿尔！有个叫巴尼的人打电话给你！”

口袋中的战争

宇宙历0079年12月25日清晨，度过了欢乐的圣诞夜后，卫星的居民们早已将先前的恐惧抛到了九霄云外，何况种种迹象表明联邦和吉恩两军之间的战争很快就要结束了，这更让人们沉醉在幸福中，忘却一切烦恼，迎接新的一年的到来。阿尔和他的母亲亦到宇宙港去迎接回家的父亲，从父亲的口中得知就在不久前联邦军击沉了一艘驶向第6区域的吉恩军巡航舰，奇怪的是那艘船舰上竟然还搭载着核弹头！阿尔闻言，忙不顾父母惊讶的呼喊急急离开。因为此时此刻，不再逃避的巴尼正驾驶着与阿尔合力修复的扎古向Alex挑战！然而，命运如此捉弄人，到了最后巴尼鼓起勇气正视自己、豁出一切而赌上的这场战斗竟然毫无意义！当阿尔赶到森林边缘地带时，战斗已经接近尾声。伤痕累累的扎古和高达相持而立，双方分别量出了热能斧和激光剑，无声中两架MS同时扑向对方，扬起的热能斧高高地落下砍去了高达的头，而后者的激光剑深深地刺入了扎古的驾驶舱……爆炸的热风将一旁观战的阿尔吹到了树下，他呆呆地望着那惨烈的景象，当看到随后赶来的救生人员将克莉丝抬出Alex的驾驶舱时，已经说不出任何语言，只是睁大着他那充满恐惧、悲伤、惊讶等各种情绪交杂的眼睛，默默地看着……“阿尔，听好！这个包裹里有我录的证言录像带，以及证据文件。为何这个卫星会变成核弹的目标，据我所知的都说了。万一我死的话把这个交到警察局，如果大人们相信这是事实的话，我想这个卫星会得救的。我本想自己直接去警察局自首，怎么说呢？那样做……我总觉得在逃避，在这里不作战的话，好象自己会变得不象自己。并不是恨联邦或要报队长他们的仇。无法表达得更好，只是想和高达打一仗，或许我是军人的关系，理由连我自己也不清楚。阿尔，我大概会死，不要因此而恨联邦士兵或高达的驾驶员，他们也只是和我一样，做他们认为该做的事而已。希望你不要怨恨他人或责怪自己，这就是我最后的请求。如果运气好活下来，而且战争结束后，我一定会回到这个卫星的，这是约定。就这样分别了，再见了！阿尔，健康地生活呦！向克莉丝问好！”看过巴尼留下的摄像机用磁碟中的内容后，阿尔躺在床上发呆。“怎么了，阿尔？”他的母亲走了进来。“做了一场梦。”阿尔答道。新的一年开始了，地球联邦军与吉恩公国签订和平条约，吉恩军宣布战败。在上学的路上，阿尔遇到了手上绑着绷带的克莉丝。克莉丝一家因为她的调职又要搬到地球去，克莉丝来向阿尔告别并让他将此事转告巴尼。“巴尼他……”泪水在眼中打了好几个转，阿尔终于忍住没有哭出来，只是低下头道：“……他也一定觉得很遗憾的。”阳光照耀在被炸掉一半的学校大楼上，操场上的孩子们排列整齐地听着校长的演讲，只不过没有多少人会将其听进去：“漫长痛苦的战争终于结束了，和平的日子到了。但是在这个殖民卫星，战争也留下了很深的伤痕。我们并不只是失去了校舍，父母、兄弟以及友人，这个和平是由许多人的牺牲所换来的。我要你们不要忘记这件事。然后当你们长大时……”再也忍不住的阿尔终于放声大哭起来，误解了他的好友们上前安慰他：“阿尔，不要哭了，战争马上会再开始的。那时一定是更、更夸张的，而且又好玩又壮大的呀！”“对呀，然后不只是弹壳，还可以拣实弹，或许还能拣到军队的枪呢！”
U.C.0080年在一片欢乐中来临了，至于三年后重新点燃的战争之火以及导致了之后更大的动乱，孩子们的无心之言也会有比大人们更正确的时候。
