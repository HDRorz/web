时代背景

U.C.0080年，一年战争终于结束了。然而，吉恩公国（又译“吉翁公国”）中仍有不甘心失败命运，一心与地球联邦敌对的人物存在。就在吉恩残兵大量被放逐到小行星阿克西斯的基地时，吉恩的迪拉兹中将，秘密召集了不愿前往阿克西斯的公国军残党，组成迪拉兹舰队，并于0080年3月，来到地月之间的暗礁宙域，开始在这里营建其作为反联邦据点的军事基地“茨之园”。
U.C.0081年5月，扎比家第三子多兹鲁的遗孀泽娜在阿克西斯病亡，其年幼的女儿美妮瓦•扎比成为扎比家最后的血脉。
8月15日，迪拉兹借吉恩公国国庆节之机开始掀起暴乱活动。
为预防战乱再次到来，联邦议会于10月13日表决通过了联邦军重建计划。作为重建计划的一环，10月20日，在乔•考文中将的主持下，由当时联邦最大的兵工企业，总部位于月面的阿纳海姆电子技术公司（A.E）展开研制尖端技术MS的高达开发计划。
11月，迪拉兹集团与阿克西斯确认共同战线。
U.C.0082年4月，联邦军决定设立NEW TYPE研究机构。
5月，殖民卫星再生计划展开，SIDE3重建工作开始。
U.C.0083年1月，迪拉兹集团开始新MS的开发，随后获得联邦军开发新型高达的情报。3月，迪拉兹集团开始制订反联邦作战计划，至7月底，星尘作战计划正式立案。
8月初，小行星阿克西斯的领导人马哈拉加•卡恩死亡，其女（年方16岁）哈曼•卡恩出任幼主美妮瓦•扎比的摄政官，并与迪拉兹签订互援协约。
9月底，阿纳海姆在月面都市冯•布拉文的工厂制造的新型RX-78高达GP-01与GP-02完工，并运往地球澳大利亚的特林敦基地。

故事发展

浦木宏（コウ•ウラキ），年青的地球联邦军少尉，配属于特林敦基地担任试飞员的工作，这里正在秘密进行着高达开发计划最后的试验阶毁，阿纳海姆电子公司设计制造的这两台高达，在性能上，都各有自己的特色，1号机装着轻型高性能装甲，以机动性为主，2号机装甲厚重，搭载力与推进力强劲，而且装备着核武器。一切都在顺利进行，孰料10月13日夜，特林敦基地突然遭到来历不明的MS部队的突袭。而且，GP-02被夺走。
启动GP-02的敌人，是迪拉兹的忠诚部下阿纳贝尔•卡多，此人在一年战争的吉恩军中，在所罗门撤退时他率领所属部队担任断后任务并给予联邦军极大的损害，因此被联邦军士兵称为所罗门的噩梦。此次于特林敦基地，他与迪拉兹的特攻部队里应外合，强夺GP-02，虽然在作战中遭遇到浦木宏驾驶的GP-01的阻截，但战斗经验丰富的卡多，很轻易地摆脱了浦木宏的追击。
GP-02设计上可搭载核弹这一点，明显是违反了《南极条约》的，联邦政府对此当然十分担忧，但同时，他们并不将迪拉兹军放在眼里，以为他们只是小股流窜的部队。
驻守于特林敦的战舰阿鲁比昂，接到了夺取GP-02的指示，以防GP-02脱离地球被用于空间战争中。
妮娜•帕普顿是阿纳海姆社的系统工程师，GP-02被强夺后，她与浦木宏等人，一同加入阿鲁比昂的追击队，一直追踪GP-02来到了非洲，经过多次遭遇战，仍未成功回收GP-02，但是她与浦木宏之间已经萌发了情愫。一行人随后追向太空，却遇上了希玛，迪拉兹军的激进分子，GP-02强夺计划完成后，她包揽下袭击追随而来的阿鲁比昂的任务，而初次来到宇宙的浦木宏，在GP-01地面装备模式下不适于宇宙战的问题上而于妮娜产生了摩擦，而作为高达GP-01与GP-02总设计师设计及系统工程师的妮娜存心刺激浦木宏，扬言给GP-01更换Fb部件（FULL BURNERN全方位推进装备，后面有详细介绍）后让浦木宏的死对头蒙夏中尉做测试机师。为了证明自己是全身心投入驾驶高达，而并非妮娜所说的“以驾驶玩具的心态驾驶高达”，在阿鲁比昂受到希玛舰队攻击时，浦木宏违反巴宁格大尉命令，强行驾驶GP-01出击，结果因为机体和机师都不适应宇宙战，而被希玛发现弱点后猛攻而大破，最后在垂死关头被巴宁格大尉救下。
为了维修被击伤的GP-01以及补给物资，阿鲁比昂前往月面都市冯•布拉文，阿纳海姆的总部。妮娜与开发部门的女孩们，正忙于将GP-01改装成宇宙形态的Fb型，沮丧的浦木宏被蒙夏中尉嘲弄后，离开部队下落不明。
凯利，一年战争时吉恩的MS驾驶员，因失去了左臂而无法再驾驶MS，战后，他在布拉文的维修工厂里偷偷修复了一年战争时期使用的机动装甲，意图参加星尘作战，一日晚，他救起了因消沉而自暴自弃的浦木宏，浦木宏被这位执着、铁一般的男子所感染，帮助他修复机动装甲，后返回了部队。阿鲁比昂离开月面时，凯利为了得到吉恩的认可，驾驶MA与浦木宏的PG-01展开决斗，被决意后的浦木宏所击败。本该在月面与阿鲁比昂和浦木宏分开的妮娜在最后关头留了下来，两人的感情更深了。
追击GP-02失败的阿鲁比昂，被编入康培宇宙岛（即所罗门）联邦军的护卫舰队，此外，地球圈外联邦的主力舰队，也都集结于此，欲以声势浩大的舰队，来震摄迪拉兹。
对于地球联邦的蔑视，迪拉兹开始还以颜色，除了继续星尘作战计划之外，卡多的GP-02装备了MK82核弹头，突破联邦舰队的防御，到达康培岛中央，照准了联邦军的旗舰。
0083年11月10日 白色核爆的闪光中，地球联邦的主力舰队全灭，卡多在脱离战线的途中与浦木宏遭遇，战斗中1号机与2号机同归于尽。经此一役，卡多“所罗门之恶梦”之名震动四方。同时，在营建与修复中的宇宙殖民卫星SIDE1与SIDE3亦遭到袭击，希玛舰队使两个殖民卫星改变轨道并相撞，互相弹开后SIDE3飞向了另一目标——月球。
（一年战争中，曾经有吉恩的部队将宇宙殖民卫星坠落于地球的作法，其带来的后果不亚于用核武器造成的危害，所谓的星尘作战计划便是以相同的手段来对付地球联邦政府。）
阿鲁比昂作为对抗星尘作战的部队，驶向宇宙空间站拉比昂洛兹补给军需并提取装备了宇宙用机动装甲的GP-03（GP-03D），同时，迪拉兹也为卡多准备了机动装甲诺亚•基路，于是一场围绕着殖民卫星落下的攻防战正式展开。
0083年11月11日 在战斗中得以成长的浦木宏，单独驾驶着GP-03，进入了最后的战域。在浦木宏赶赴战场的过程中卫星掉向月球，月球一方在危机关头接受了吉恩的“建议”（“星屑”计划的一部分），启动了卫星的推进器，使SIDE3飞离了月球，驶向“星屑”的正真目的——地球，联邦基地贾布罗！
0083年11月12日21:47 地球联邦为了消灭卫星，使用太阳能系统Ⅱ（曾在一年战争中出现）向卫星照射，失败。太阳能系统部分被卡多的诺亚破坏。
22:41 希玛与浦木宏交战，希玛战死。
确保着殖民卫星飞向地球的卡多，除了对付联邦军的联合舰队之外，亦要清除发动了政变的希玛舰队，而浦木宏的攻击，使他无法脱身；卡多来到SIDE3的控制室中进行最后的轨道调整时遇到了浦木宏，并被浦木宏用枪击中，然而妮娜在同时面对浦木宏与卡多时，将以往的恋人——卡多，从浦木宏手中救走，浦木宏颤抖地质问妮娜对自己的感情，妮娜无言以对，这使浦木宏万念俱灰。
U.C.0083年11月13日00:00 卡多与浦木宏在战场中再次相遇，千万种感情交织，浦木宏冲向卡多，开始了最后一次宿命的决斗。正当他们激战正酣时，太阳能系统Ⅱ的余下部分开始二次照射，打断了两人的决斗，但是依然无法阻止卫星的下坠。
终于，殖民卫星仍然还是于11月13日00:34飞过贾布罗上空，坠落于北美！而卡多也在与浦木宏的最后一次决斗被打断后在乱战中被击落。
11月23日，军事法庭召开审判，处分在迪拉兹之乱中犯下各种过失的军内人员，其中，浦木宏少尉被判刑一年。
12月4日，联邦军在加米托夫•海曼准将的建议下，组成了追击、搜捕吉恩残党及反联邦分子的治安部队“提担斯”，对反地球分子展开了严厉的讨伐。
U.C. 0084年3月10日，由于事关太多不宜泄露的机密，卫星落下的真相、以及GP系列高达开发计划，被官方从正式记录上删除，而浦木宏的罪状，也随之被撤消……
尾声：南美联邦基地外，浦木宏站在被卫星撞击烧焦的麦田中伫立，偶然地邂逅了好友吉斯，还有，妮娜。

历史进程

事端

一年战争已告一段落的三年后，基于地球联邦军重建计划而被提议的“高达开发计划”开始执行，其中的两台原型机RX-78GP01与RX-78GP02A为了实施性能测试而搬运至澳大利亚"托林顿"基地，但是吉恩残党查知此项计划，并且袭击该基地企图夺取装载有核弹的GP02A。曾经被称呼为“所罗门的恶梦”的吉恩军王牌机师阿纳贝尔•卡多少校（又译为阿纳贝鲁•卡图）成功的搭上了GP02A，但是在企图脱离的GP02A面前，新手测试驾驶员浦木宏驾驶了GP01拦阻，揭开了宇宙世纪0083年迪拉兹纷争的序幕。

发展

夺取GP02的卡多前往非洲金巴莱德矿山基地，在当地残党军领袖诺巖•彼达少将的协助下成功以HLV脱离地球，回到暗礁空域，由埃丘•迪拉兹中将所领导的吉恩残党（迪拉兹舰队）基地。并纠集地球圈吉恩军残存势力向地球联邦宣战，作战代号为“星尘”。其后卡多以GP02A袭击联邦军于孔贝岛（旧吉恩军所罗门要塞）宙域举行的观舰式（阅兵典礼）并以核弹摧毁了2/3的联邦舰队，战斗中GP01Fb与GP02同归于尽。同时西玛•卡拉豪中校所率领的海军陆战部队劫持两枚太空殖民卫星并使其中一枚向月球落下，而后借由月面都市冯•布朗的设施令其再改变轨道朝向地球而去。
正当迪拉兹舰队为使殖民卫星顺利落下，准备与联邦军轨道机动舰队决战时，西玛叛变并杀害迪拉兹。联邦军方的鹰派也最终冒了出来。在能够在改变殖民卫星轨道以阻止殖民卫星落下的情况下，却下令部队停止一切攻击活动。试图以SOLAR SYSTEM II系统减小殖民卫星并改变其航线，在对军方总指挥部不构成威胁的情况下，使殖民卫星撞击地球，以打击吉恩宣布投降后，联邦内部崛起的温和派成员。卡多则率领残余部队保护殖民卫星突破联邦军所布置的SOLAR SYSTEM II以及浦木所驾驶的RX78-GP03阵线。在UC0083年11月13日凌晨00:34分、殖民卫星最终撞入地球、但是没有像联邦军预期的那样撞到总部加布罗，而是落在北美洲平原谷仓地带。殖民卫星下落后联邦利用新的恐慌如愿建立了泰坦斯，控制了军队。

结果

U.C.0083年12月04日、地球联邦军内激进派系泰坦斯发表成立宣言、为后继的机动战士Z高达构成了舞台。
阿鲁比昂被指派新任舰长、艾帕•席那普斯舰长因作战中反抗巴斯克•欧姆的命令而被处决。
亚纳海姆电子冯•布朗基地的负责人因为提供AGX-04（RX78-GP04）给西玛以及转变殖民卫星动向，害怕被联邦政府追究而自杀。（一说为泰坦斯所暗杀）
浦木弘被军事法庭判处一年有期徒刑。
U.C.0084年03月10日、RX78-GP系列开发计划被抹消，随即，浦木宏的罪状被撤销。

机动战士高达0083 吉恩的残光（日文名：机动戦士ガンダム0083 ジオンの残光）于1992年在日本公映。
