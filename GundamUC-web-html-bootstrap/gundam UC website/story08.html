简介

《高达UC》作为最新的宇宙世纪动漫化作品，其主角机为继承了GUNDAM之名的 RX-0 独角兽高达。《高达UC》，中文全称为《机动战士高达独角兽》，日文原名为「机动戦士ガンダムUC（ユニコーン）」。
其中“UC”应读作“UNICORN”，即“独角兽”，日语汉字为“一角獣”；更深层地，“UC”也代表着“Universal Century”，即“宇宙世纪”，也就是由富野由悠季创作的初代TV动画《机动战士高达》的世界观，目前“高达系列”公认的正统系列。
《高达UC》原为福井晴敏执笔的小说，在高达30周年纪念时（2009）公布动画化，后再公布为2010开始发售的OVA（预定全6话，每话近60分钟，但是官方于2012年5月13日宣布将追加第七话为最终话）。
《高达UC》的故事发生在U.C.0096年，即“第二次新吉翁战争（夏亚的反乱）”的三年后。

OVA分集剧情（更新中）

第一话「独角兽之日」

宇宙世纪0096——。引发新吉翁与地球联邦军之间战火的事件“夏亚的逆袭”过后的第三年。
住在工业殖民地「工业七号（Industrial 7）」中的少年巴纳吉·林克斯（Banagher Links）与一名自称是奥黛丽·本（Audrey Burne）的神秘少女相遇。奥黛丽声称她正在阻止毕斯特财团将「拉普拉斯（Laplace）之盒」转交给被称为「袖章」的新吉翁残党，这一将会引发一场新战争的行为，巴纳吉决定帮助她。然而殖民地在「袖章」和同样想阻止交接的地球联邦军之间的战斗爆发后变成一片战场。正当被卷入战火的巴纳吉寻找奥黛丽时，他与「拉普拉斯之盒」的钥匙——机动战士独角兽（Unicorn）命中注定般地相遇了。
　　 （2010.2.20 PSN在线配信 3.24正式发售BD&DVD）

第二话「赤色彗星」

身为毕斯特财团家主的卡帝亚斯（Cardeas），在临终前将作为「拉普拉斯之盒」的钥匙的机动战士——独角兽托付给了年轻的巴纳吉。在与「袖章」驾驶员——玛莉妲·库鲁兹（Marida Cruz）和她的刹帝利（Kashatriya）交战后，巴纳吉被隆德·贝尔（Londo Bell）所属的攻击型登陆舰「拟·阿卡马（Nahel Argama）」所捕获。同样在舰上的还有巴纳吉的朋友拓也（Takuya）和米寇特（Micott）以及为试图阻止「盒子」的交接而与卡帝亚斯接触的少女——奥黛丽·本。
敌方的一台MS单独对「拟·阿卡马」发动了攻击。这台红色的MS就是新安州（Sinanju），而他的驾驶者弗尔·伏朗特（Full Front al）——作为「袖章」的领袖，同时也是被称为“夏亚再世”的男人。
　　 （2010.10.30 PSN在线配信 11.12正式发售BD&DVD）

第三话「拉普拉斯的亡灵」

在因为保护米妮瓦（Mineva）而发动独角兽Gundam后，巴纳吉在与弗尔·伏朗特的新安州的战斗中被捕获。他现在已经被带到资源卫星帕劳（Palau）——被称为「袖章」的新吉翁残党的大本营，并滞留在那里。
为了夺回巴纳吉和独角兽，联邦宇宙军特种作战群的塔克萨（Daguza）中校配合「拟·阿卡马」对帕劳发动了袭击。巴纳吉趁乱带着独角兽逃出了帕劳，然而…
为了发现会随着NT-D系统的启动而被揭开的指向「拉普拉斯之盒」的路线的弗尔·伏朗特，派来了玛莉妲和她的刹帝利，挡住了巴纳吉的去路。
　　 （2011.3.5 PSN在线配信 4.7正式发售BD&DVD）

第四话「在重力的井底」

围绕着「拉普拉斯之盒」的战斗扩散到了地球，甚至连地球联邦政府的首都——达喀尔（Dakar）都正在遭受战争的恐怖。地球联邦政府的中心人物——罗南·马瑟纳斯（Ronan Marcenas）为了阻止「盒子」的开启而与隆德·贝尔的司令布莱特·诺亚（Bright Noa）接触。
与此同时，降落在地球的巴纳吉，正前往特林顿（Torrington）基地。在那里，有关「盒子」的新情报将被揭露。然而吉翁残党的愤怒将驱使MA尚布罗（Shamblo）的驾驶者罗妮（Loni）将战争变成了一场屠杀。
（2011.11.12 PSN在线配信 12.2 正式发售BD&DVD）

第五话「黑色的独角兽」

巴纳吉受到黑色的机动战士“报丧女妖（Banshee）”攻击后被监禁。在无路可逃的情况下，巴纳吉仍然坚持封印独角兽Gundam揭示的指向「拉普拉斯之盒」的新坐标，这样的他，让隆德贝尔队的司令布莱特看出了巴纳吉身上有历代Gundam驾驶员所显现的矜持与可能性，同时他向曾今与他共同战斗过的老朋友那里寻求帮助。
另一方面，从罗南引渡至玛莎（Martha）的米妮瓦，作为让巴纳吉能够听命行事的棋子被移送至超大型运输机「迦楼达（GARUDA）」上。围绕「盒子」的战斗移向高空并最终回到宇宙…
（2012.5.19收费在线开始 6.8式发售BD&DVD）

第六话「宇宙与地球」

由于弗尔·伏朗特的介入而得以逃离地球联邦军舰队追击的「拟·阿卡马」，被迫接受与「袖章」并肩作战。另一方面，伏朗特揭露了他用来对付联邦的密策并企图问出指向「拉普拉斯之盒」的下一个座标；而巴纳吉与米妮瓦，对他娓娓道出的未来竟是如此地冷酷无情感到毛骨悚然。此时，一艘联邦军的巡逻舰正在接近「拟·阿卡马」。究竟是要击沉巡逻舰呢？还是要毁弃与「袖章」之间的同盟呢…？在敌我双方夹缝间苦恼的巴纳吉一伙人，最后的选择是？
（2013.3.2收费在线观看、剧场限定BD发售开始 3.22正式发售BD&DVD）
